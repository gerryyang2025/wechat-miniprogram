## 1. 文档职责

本文档用于统一解释 `private-domain-operation` 中出现的缩写、英文术语和容易歧义的专有名称，减少跨文档重复解释。

## 2. 产品与设计术语

- `IP`
  - `Intellectual Property`
  - 在当前项目语境下，更接近 `个人品牌身份 / 个人品牌资产`，即“个人 IP”
- `IA`
  - `Information Architecture`
  - 信息架构，通常指页面结构、导航层级、页面关系与信息组织方式
- `MVP`
  - `Minimum Viable Product`
  - 最小可行产品，强调用最小范围验证核心业务闭环
- `CTA`
  - `Call To Action`
  - 行动引导按钮或行动引导区域，例如“立即购买”“立即报名”
- `P0`
  - `Priority 0`
  - 必须进入一期的最高优先级内容
- `P1`
  - `Priority 1`
  - 建议进入一期，但可作为增强项或小版本补充
- `Out`
  - `Out of Scope`
  - 明确不进入当前一期范围的功能

## 3. 运营与业务术语

- `CRM`
  - `Customer Relationship Management`
  - 客户关系管理
- `SCRM`
  - `Social Customer Relationship Management`
  - 社交化客户关系管理，强调社群、私域、传播链路和社交触达
- `GMV`
  - `Gross Merchandise Volume`
  - 商品交易总额，常用来衡量交易规模
- `SKU`
  - `Stock Keeping Unit`
  - 库存量单位，常用于区分同一商品的不同规格、价格或版本

## 4. 平台与技术术语

- `API`
  - `Application Programming Interface`
  - 应用程序编程接口
- `SDK`
  - `Software Development Kit`
  - 软件开发工具包
- `H5`
  - `HTML5 Page`
  - 常用于指移动端网页、活动承接页或浏览器页面
- `PC`
  - `Personal Computer`
  - 这里通常指桌面端后台管理台
- `App`
  - `Application`
  - 移动应用程序
- `AI`
  - `Artificial Intelligence`
  - 人工智能
- `HTTP`
  - `HyperText Transfer Protocol`
  - 超文本传输协议
- `JSON`
  - `JavaScript Object Notation`
  - 常见的数据交换格式
- `TTL`
  - `Time To Live`
  - 存活时间，常用于缓存、会话或短期状态失效控制
- `ACID`
  - `Atomicity, Consistency, Isolation, Durability`
  - 事务的四个核心特性：原子性、一致性、隔离性、持久性

## 5. 后端与存储术语

- `Go`
  - Google 推出的编程语言，也常写作 `Golang`
- `Gin`
  - Go 生态中的高性能 Web 框架
- `MySQL`
  - 常用关系型数据库管理系统
- `etcd`
  - 分布式键值存储系统，适合配置、轻量元数据、锁和协调状态
- `COS`
  - `Cloud Object Storage`
  - 对象存储，适合视频、音频、图片、附件等大文件
- `VOD`
  - `Video on Demand`
  - 云点播，适合视频上传、转码、回放分发和播放优化
