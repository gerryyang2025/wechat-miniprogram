# Code of Conduct

This project aims to be welcoming, practical, and respectful.

## Our Standard

Participants are expected to:

- Be respectful and constructive
- Assume good intent while giving direct feedback
- Focus on the work, not the person
- Help make onboarding easier for new contributors
- Keep discussions professional and inclusive

## Unacceptable Behavior

The following behavior is not acceptable:

- Harassment, intimidation, or personal attacks
- Discriminatory language or exclusionary behavior
- Repeatedly derailing technical discussions
- Publishing private information without permission
- Hostile or insulting review comments

## Enforcement

Project maintainers may edit, remove, or reject comments, issues, pull requests, or other contributions that do not align with this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, contact the project maintainer through a private channel when possible. Avoid escalating sensitive conflicts in public threads unless necessary for safety.
